----------- HANOI PROBLEM SOLVING ALGORITHM WITH CONSOLE GRAPHICS --------------

For Windows only (using windows.h).

Usage:

 Hanoi [arguments]

 Program settings:
	Argument        Meaning         			Example
	-obr            Set obr. count (1-10)                   -obr 8
 	-spcr           Set creation speed (period in ms)     	-spcr 100
 	-spmv           Set moving speed (period in ms)         -spmv 100

 Special effects:
	Argument        Meaning         			Example
 	-youwon         Try :-)                                 -youwon
 	-obrrc          Obruc random color                      -obrrc
 	-beep           Beep                                    -beep
 	-bilik          Try :-)                                 -bilik
